# Smart Water Management – Lake Water Level Forecasting
Time series analysis and forecasting of lake water levels using environmental and meteorological data.

## Overview

This project analyzes environmental and meteorological data to forecast lake water levels as part of a smart water management system. Time series analysis and machine learning techniques were used to identify patterns, understand relationships between variables, and support water resource management.

---

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Time series analysis
- Correlation analysis
- Feature engineering
- Water level forecasting
- Model evaluation

---

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Scikit-learn
- Statsmodels

---

## Dataset

The project uses the ACEA Lake dataset containing daily environmental measurements collected between **2002 and 2020**, including rainfall, temperature, lake level, and water flow.

https://www.kaggle.com/competitions/acea-water-prediction/data
